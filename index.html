Download the documentation of The Laptops/Repair Clinic software:
The Repair Clinic (TRC) application onboarded all the different applications used by the TRC into one application rather than the usual of using multiple applications at the same time and shuffling between them.
• It provided an interface for employees to login, register, retrieve forgotten details, email, set calendar appointments, store & contact customers and record & print financial records.
• Microsoft Access & SQL was used for data management system. Java and Visual Basic for application building.
